java.net.URL Timeouts

Adrian Sutton

August 26, 2008

Sylvain Wallez:

If your application uses java.net.URL, and chances it does are very high, and you are using Sun’s JVM (since 1.4.2), you should set the sun.net.client.defaultConnectTimeout and sun.net.client.defaultReadTimeout system properties to a reasonable value. Otherwise, if a remote site hangs, your application or server will also hang. Useful to know…
